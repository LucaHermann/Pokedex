# Pokedex

## Description

This project is a Pokedex built with React, Apollo Client for GraphQL queries, and React Query for data fetching. It allows users to browse and view details of Pok√©mon.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/pokedex.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```
